# 6주차 과제

## 1. 댓글 기능

- 게시글에 댓글을 작성, 수정, 삭제, 조회할 수 있어야 합니다.
- 댓글은 300자 이내로 제한되며, 게시글 상세 조회 시 함께 내려갑니다.
- 댓글 작성자는 유저 정보와 연결되어야 합니다.
